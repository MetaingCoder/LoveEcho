# 💌 LoveEcho 产品介绍

> **让每一次心动，都有迹可循。**

**LoveEcho** 是一款专为情侣打造的**纪念日追踪与情感记录**应用。它不仅仅是一个计时器，更是一个**情感记忆库**，通过**实时计时系统**与**情感可视化**功能，帮助你们珍藏恋爱历程中的每一个重要瞬间，打造专属的浪漫空间。

---

### ✨ 核心亮点

- **⏳ 动态纪念日计时器**
    - **精确到秒**的实时计时系统，记录恋爱天数
    - 支持**自定义起始日期**，适应不同情侣的纪念需求
    - **响应式设计**，在各种设备上完美显示
- **📊 情感状态可视化**
    - **双维度状态展示**：当前情绪状态与关系健康度
    - **动态图表**直观呈现情感变化趋势
    - **自定义状态标签**，让表达更个性化
- **👫 情侣档案系统**
    - **对称式双人头像展示**，体现平等与和谐
    - **昵称自定义**功能，增添专属感
    - **故事时间线**，记录重要里程碑
